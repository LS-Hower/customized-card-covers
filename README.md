# customized-card-covers

定制硬质塑料卡套时使用的图片

## 构建

```bash
make
```

## 参考颜色

（ `#` 的前面有一个空格，是为了在 VS Code 里实时预览颜色）

- ` #003399` 北邮蓝
- ` #880000` 绀珠传红

微软画图软件默认颜色：

- ` #FFF200` 黄色
- ` #FFAEC9` 粉色

ISO C++ 标志：

- ` #659AD2` 左上（浅）
- ` #00599C` 右（中）
- ` #004482` 下（深）

ISO C++ 标志（红色色调，即 H = 0）：

- ` #D26565` 左上（浅）
- ` #9C0000` 右（中）
- ` #820000` 下（深）
